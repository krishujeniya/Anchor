# Security Policy

## Supported Versions

Currently, only the `main` branch (latest release) is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within ANCHOR (for example, a way to bypass the sandbox rules or arbitrary file execution outside `.agents/`), please DO NOT open a public issue.

Instead, please email the maintainers privately. We will acknowledge your report within 48 hours and work with you to patch the vulnerability safely before public disclosure.
